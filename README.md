# Customer Churn Prediction

## Project Description
The **Customer Churn Prediction** project aims to predict which customers are likely to leave a service or company. It uses **Python** and **Machine Learning** to analyze customer data and build classification models that help businesses make better decisions to retain customers.

---

## Features
- Customer data analysis.  
- Building classification models to predict customer churn.  
- Uses Python libraries such as `scikit-learn`, `pandas`, and `numpy`.  
- Models can be reused for new customer data.

---

## Key Files
- `app.py` : Main application file.  
- `churn_prediction_model.ipynb` : Notebook for data analysis and model building.  
- `dataset/` : Contains training and testing data files.  
- `inference.py` : Script for predicting new customer churn using trained models.  
- `.joblib` files : Saved models and preprocessing
